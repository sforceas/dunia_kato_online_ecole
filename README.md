# Dunia Kato Online E-cole
Online e-learning platform from Dunia Kato Senegal.

## Stack

### Frontend
- HTML 5
- CSS 3
- JavaScript
- React.js

### Backend (API REST)
- Python 3.10
- Django 4.0

## DDBB
- PostgreSQL 13.4

### Deployment
- Docker 20.10.11
